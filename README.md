# Motor Risk Underwriting & Claim Cost Dashboard 🚗

This interactive Streamlit dashboard is designed for the **Chief Underwriting Officer** to analyze, interpret, and action risk-ranking models on renewal policies. It shifts a highly technical predictive claim cost model into a production-grade, intuitive tool for commercial and private underwriters.

## 🛠️ Tech Stack & Models Used
- **Deployment Platform:** Streamlit Community Cloud
- **Core Model:** Poisson GLM (General Linear Model) - Champion Model
- **Baseline Models:** Random Forest (Shallow), XGBoost (Shallow), Baseline Mean Predictor
- **Visualizations:** Plotly Express & Graph Objects

## 📂 File Directory
- `app.py` - Core Streamlit application script containing the UI logic and risk calculation scoring engine.
- `requirements.txt` - Required library list for deployment.
